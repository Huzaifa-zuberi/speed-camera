How to get a 2captcha API key:
  1. Go to 2captcha.com and register an account
  2. Deposit funds (minimum $1 = ~1000 solves)
  3. Go to your API settings and copy your API key
  4. Paste it above and click Save
  5. Each vehicle lookup will cost ~$0.001 (1/10th of a cent)

Once configured, the system will automatically solve the Sindh excise portal reCAPTCHA and fetch real vehicle registration data for K-series (Karachi) plates. Falls back to mock data if the lookup fails or the key is not set.
